# サービス比較：どの課題にどのサービスが向いているか

このページでは、アーガイルの主要4サービスの役割を一覧で整理します。

---

## ざっくりマッピング

- **AIアシスタント**  
  → 社内の生成AI活用を広く整えたい / PoC から始めたい

- **生成AI受託開発**  
  → 既存システムと深く連携した本番システムを作りたい

- **TagLive**  
  → 観光・商業施設・イベントで SNS 投稿（UGC）を可視化したい

- **コピーロボット**  
  → キャラクターAI・IP活用・対話型コンテンツを作りたい

---

## 比較の観点（例）

- 想定ユーザー：社内ユーザー / 一般ユーザー / ファン  
- 導入スピード：SaaS / PoC / フルスクラッチ  
- 必要な社内リソース：IT 部門 / 企画部門 / 現場運用  
- 費用感：ライト〜エンタープライズ

※具体的な料金や個別事例は、HTML 正規ページと各サービス公式サイトを参照してください。
