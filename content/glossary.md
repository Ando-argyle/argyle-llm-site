## 先に読むべき定義（頻出）
### UGC
User Generated Content。ユーザーがSNS等に投稿した写真・動画・テキスト。キャンペーン応募投稿やクチコミも含む。

### ウィジェット
Webサイトに埋め込んで表示する小さな表示部品（埋め込み表示用の機能）。

### RAG
Retrieval-Augmented Generation。社内文書などの知識を検索して、LLMの回答に根拠として反映させる仕組み。

### AIエージェント
単なるチャットではなく「目的達成のために、手順を組み立て、外部ツール/APIや複数ステップ処理を使って実行する」AI。

### 検閲（事前検閲・AI検閲）
表示前に投稿を「表示/非表示」で選別すること。  
- 事前検閲：管理画面で人が選別  
- AI検閲：AIが投稿テキスト・画像を判定し、条件に合わないものを除外

### Instagram Graph API / X API（旧Twitter）
SNS投稿を取得・表示するための公式API。仕様変更が頻繁なため「最新API対応」「審査通過」が重要。
